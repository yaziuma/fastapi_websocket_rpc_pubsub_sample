Job: 相手サーバへ処理を依頼する

1対1 の命令です。相手サーバにジョブをキュー登録し、処理種別と優先度とパラメータを返します。

例: order_id や warehouse など、処理対象を JSON で渡します。

Alert: 相手 callback へ緊急通知する

1対1 の割り込みです。相手サーバではなく、相手が公開している callback 側に直接アラートを差し込みます。

PubSub: 出来事を全体へ配信する

1対N のイベント共有です。送信側は購読者を意識せず、topic と event_type と payload だけを publish します。

Topic は系統、Event Type は変化種別、payload は共有したい状態変化そのものです。
